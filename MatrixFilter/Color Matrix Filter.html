Color Matrix Filter

Color Matrix Filter--颜色矩阵滤镜，它是Adobe公司flash出的一个滤镜算法。可以在颗粒等级上提供更好的控制方法。

Color Matrix Filter为四行五列的多维矩阵，如图所示

￼

RGBA分别代表红色通道、绿色通道、蓝色通道和alpha通道(不透明度，1是完全不透明，0是完全透明)，offset表示偏移量。计算公式如下：

redResult   = a[0] * srcR + a[1] * srcG + a[2] * srcB + a[3] * srcA + a[4]
greenResult = a[5] * srcR + a[6] * srcG + a[7] * srcB + a[8] * srcA + a[9]
blueResult  = a[10] * srcR + a[11] * srcG + a[12] * srcB + a[13] * srcA + a[14]
alphaResult = a[15] * srcR + a[16] * srcG + a[17] * srcB + a[18] * srcA + a[19]

简单解释一下，公式中srcR、srcG、srcB、srcA分别是你原图中红色、绿色、蓝色和alpha值，第一行的redResult则是通过矩阵计算改变后的你图片中的红色数值。比如上图中a[0]为1，a[1]、a[2]、a[3]、a[4]都为0，则redResult = srcR，也就是说红色数值没有发生改变。以此类推可以发现，greenResult、blueResult、alphaResult都跟原图保持一致，也就是说上图的矩阵没有对原图做任何修改。
